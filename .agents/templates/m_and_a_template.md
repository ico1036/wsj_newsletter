# M&A 거래 템플릿

## 구조

```markdown
### [N]. [기업명 A], [기업명 B] [거래액] 인수 ⭐ [주요 거래 여부]

**거래 구조**
- **총 거래 가액**: [검증된 금액] (부채 포함 여부)
- **주당 가격**: [검증된 가격] ([기준일] 종가 대비 [프리미엄]% 프리미엄)
- **지분 구조**: [기업 A] [비율]%, [기업 B] [비율]%
- **예상 마감**: [날짜]
- **연간 매출**: 합병 후 [금액] 규모

**전략적 배경**
- [배경 설명 1]
- [배경 설명 2]
- [행동주의 투자자 영향 등]

**합병 후 포트폴리오**
- [브랜드/제품 정보]
- [시장 위치]

**리스크 요인**
1. **[리스크명]**: [설명]
   - [상세 설명]
   - [관련 정보]

**시장 반응**
- [기업 A] 주가: [변동률] ↑/↓ ([설명])
- [기업 B] 주가: [변동률] ↑/↓ ([설명])
```

## 데이터 매핑

- `key_facts.deal_value` → 총 거래 가액
- `key_facts.premium` → 프리미엄
- `key_facts.timeline` → 예상 마감
- `risks` → 리스크 요인
- `market_impact` → 시장 반응
- `quotes` → 인용문 (전략적 배경에 포함)
